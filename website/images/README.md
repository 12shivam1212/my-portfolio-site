## Image Guidelines:

- Use high-quality images (at least 72 DPI for web)
- Optimize file sizes (use tools like TinyPNG or ImageOptim)
- Consider using WebP format for better performance
- Maintain consistent aspect ratios for project images
- Ensure images look good on both light and dark backgrounds

## Tools for Creating Images:

- **Screenshots**: Use browser dev tools or screen capture tools
- **Mockups**: Figma, Adobe XD, or online mockup generators
- **Photo Editing**: Adobe Photoshop, GIMP, or online editors
- **Optimization**: TinyPNG, Squoosh, or ImageOptim
